Welcome to Moodify

What is Moodify?

Moodify is built using Spotify's web API to dynamically generate a unique playlist for each user. Once you log in, you'll be able to create a playlist using your liked music based on however you are feeling. Don't have any liked music? We have generated playlists based on pre-defined moods!

If you want to learn more about who & why this was created, please head to the About section!

Interested in learning more about the technical side? Please check out the github!
